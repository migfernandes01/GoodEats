## A restaurant review platform

### Description:

GoodEats is a platform where the users can review restaurants. Each user can submit restaurants as well as see and review others. This is my first full-stack project where I put everything I knew in practice and tested myself with problem-solving everyday.

### Used Technologies:

- HTML&CSS
- Bootstrap 5
- Javascript
- Node.js&Express
- EJS
